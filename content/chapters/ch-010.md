# Agent Factory Architecture

## Overview

The Agent Factory is the capstone architecture of this course. Instead of manually building each agent from scratch, a General Agent reads a specification document and manufactures a fully configured Custom Agent tailored to a specific job. This is how you build AI systems that scale — not by writing more code, but by writing better specs. By the end of this chapter, you will have implemented a working Agent Factory.

## Key Concepts

**The Agent Factory Pattern**
```
Spec Document → [General Agent] → Custom Agent
```
- **Spec Document**: A markdown file describing the agent's purpose, tools, behavior rules, and guardrails
- **General Agent**: Claude reading the spec and configuring a new agent
- **Custom Agent**: A fully initialized agent ready to serve its specific domain

**Why This Pattern?**
Without Agent Factory, creating a new agent requires writing code. With Agent Factory:
- A product manager writes a spec in markdown
- The factory reads it and produces a working agent
- No additional code required
- Consistent quality across all manufactured agents

**Components of an Agent Spec**
```markdown
# Agent Spec: Customer Support Bot

## Purpose
Handle tier-1 customer support for SaaS products.

## Tools
- search_kb: Search the knowledge base
- create_ticket: Create a support ticket
- escalate: Escalate to human support

## Behavior Rules
1. Always search KB before answering
2. Create ticket if issue unresolved after 3 exchanges
3. Escalate if customer expresses frustration 3x

## Guardrails
- Never share pricing information
- Never make promises about features
- Always end with: "Is there anything else I can help you with?"

## Tone
Professional, empathetic, solution-focused
```

## Deep Dive

**The General Agent (Factory Core)**
```python
import anthropic
from pathlib import Path

client = anthropic.Anthropic()

FACTORY_SYSTEM = """You are an Agent Factory. Your job is to:
1. Read the provided agent specification
2. Extract: purpose, tools, behavior rules, guardrails, tone
3. Synthesize a complete system prompt for the custom agent
4. Return ONLY the system prompt — no explanations

The system prompt must be actionable, specific, and include all rules."""

def manufacture_agent(spec_path: str) -> str:
    """Read a spec file and manufacture a system prompt for the custom agent."""
    spec = Path(spec_path).read_text(encoding="utf-8")

    response = client.messages.create(
        model="claude-sonnet-4-20250514",
        max_tokens=2048,
        system=FACTORY_SYSTEM,
        messages=[{
            "role": "user",
            "content": f"Manufacture an agent from this spec:\n\n{spec}"
        }],
    )
    return response.content[0].text

def create_agent(system_prompt: str) -> callable:
    """Create a callable agent from a manufactured system prompt."""
    def agent(user_message: str, history: list = None) -> tuple[str, list]:
        messages = (history or []) + [{"role": "user", "content": user_message}]
        response = client.messages.create(
            model="claude-sonnet-4-20250514",
            max_tokens=1024,
            system=system_prompt,
            messages=messages,
        )
        reply = response.content[0].text
        messages.append({"role": "assistant", "content": reply})
        return reply, messages
    return agent
```

**Full Agent Factory Pipeline**
```python
# Step 1: Manufacture the agent
system_prompt = manufacture_agent("specs/customer-support-bot.md")

# Step 2: Create the agent function
support_bot = create_agent(system_prompt)

# Step 3: Use the agent
reply, history = support_bot("My payment failed but I was charged twice.")
print(reply)

# Continue the conversation
reply, history = support_bot("I already tried that and it didn't work.", history)
print(reply)
```

**Agent Registry (Production Pattern)**
```python
from pathlib import Path
import json

class AgentRegistry:
    """Manages manufactured agents with caching."""

    def __init__(self, specs_dir: str, cache_file: str = "agent_cache.json"):
        self.specs_dir = Path(specs_dir)
        self.cache_file = Path(cache_file)
        self.cache = self._load_cache()

    def _load_cache(self) -> dict:
        if self.cache_file.exists():
            return json.loads(self.cache_file.read_text())
        return {}

    def get_or_manufacture(self, agent_name: str) -> str:
        """Return cached system prompt or manufacture a new one."""
        spec_path = self.specs_dir / f"{agent_name}.md"
        spec_hash = str(spec_path.stat().st_mtime)  # Use mtime as version

        if self.cache.get(agent_name, {}).get("version") == spec_hash:
            return self.cache[agent_name]["system_prompt"]

        # Spec changed or not cached — manufacture fresh
        system_prompt = manufacture_agent(str(spec_path))
        self.cache[agent_name] = {"version": spec_hash, "system_prompt": system_prompt}
        self.cache_file.write_text(json.dumps(self.cache, indent=2))
        return system_prompt

# Usage
registry = AgentRegistry("specs/agents/")
tutor_prompt = registry.get_or_manufacture("course-tutor")
tutor = create_agent(tutor_prompt)
```
